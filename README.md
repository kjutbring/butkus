# Butkus - funny quotes

Read funny Butkus quotes. Fetches data from https://butkus.xyz/api/quote (thx topcat!)

### Why?

### How?
Get the latest release and install via adb. 

See if device is connected

```
adb devices
```

Install .apk
```
adb install path/to/application.apk
```
